# RoboteraStudio 使用手册

RoboteraStudio 使用手册

面向测试人员的 RoboteraStudio 用户使用指南，涵盖登录、桌面应用与云端 Web 全部功能。

---

## 1. 快速入门

### 第一次使用流程

1. **连接网络**：连接机器人的WIFI
2. **配置xos白名单:**连接xos网站http://192.168.8.100:1888/，确认是否有白名单功能，有则配置，无则跳过
3. **启动应用** ：双击桌面图标或运行 RoboteraStudio
4. **登录** ：输入用户名和密码，或点击「游客登录」直接进入
5. **选择测试模式** ：在主界面选择「整机测试」（当前版本仅支持整机测试，故跳过）
6. **连接设备** ：等待系统自动连接 SSH 和 ROS2（连接中会显示提示）
7. **开始测试** ：在功能测试页选择测试项，点击「开始测试」
8. **完成测试** ：按顺序完成各项测试，测试结果会自动上传至云端

## 3. 桌面应用

### 3.1 界面布局

[图片已省略]

主界面顶部为导航栏，包含四个 Tab：

- **主页**：显示版本信息、基础数据接口操作
- **功能测试** ：自动化测试、模组测试、灵巧手测试（当前版本主要使用整机测试）

[图片已省略]

- **配置** ：软件信息预期值等配置

右上角显示当前登录用户，可点击「退出登录」切换用户。

### 3.2 功能测试 - 整机测试界面

功能测试页包含：

- **设备基础信息** ：SN 码、机器人型号、生产日期、SDK 版本、电池电量
- **测试项目列表** ：9 项整机测试，每项显示名称、测试方法、测试标准、测试结果
- **开始测试 / 停止测试** ：控制测试流程

### 3.3 各测试项说明

| 测试项 | 测试方法 | 测试标准 | 简要说明 |
|-|-|-|-|
| 整机状态指示灯 | 上电目视检测 | 指示灯亮灯正常、无闪烁 | 以 10Hz 发布故障状态，弹窗提示检查灯带颜色 |
| 急停遥控器 | 按键功能测试 | 急停响应及时、复位正常 | D 键模组断电 + A 键 NUC 断电，共 2 步 |
| TypeC 接口 | 插拔 + 通信测试 | 接口接触良好、通信正常 | 通过 SSH 执行 dmesg 检测 TypeC 识别 |
| 软件信息 | 版本读取校验 | 版本匹配、信息完整 | 读取 SDK / 关节版本、扭矩系数、掩码，与预期配置比对 |
| PID 参数 | 参数设置验证 | 设置值与读取值一致 | 从 CSV 读取 PID，设置后验证并自动保存 |
| 整机标零 | 标零执行验证 | position 绝对值 < 0.02 | 6 步制动器控制（开踝→关踝→开膝→关膝→开髋→关髋）+ 标零 + 结果检查 |
| 雷达 | 话题数据检测 | 话题有数据 | 订阅 /livox/lidar，5 秒内检测到数据即为成功 |
| 双目相机 | 获取相机图像 | 相机图像人为判断 | 获取相机一帧图像输出在界面 |

每项测试执行前可能有弹窗确认，请按提示操作。测试完成后结果会显示在列表中，并自动上传云端。

### 3.4 配置页

[图片已省略]

在「配置」Tab 中可编辑 **软件信息预期值** ：

1. 以 JSON 格式编辑预期配置（SDK 版本、关节版本、扭矩系数、关节掩码等）
2. 点击「保存」按钮
3. 保存成功后，软件信息测试将以此配置进行比对

### 3.5 切换用户、退出登录

- 点击右上角「退出登录」可返回登录界面，切换其他用户
- 重新登录后，测试环境会重新初始化

## 4. 云端 Web 界面

云端 Web 界面用于查看和管理测试数据，支持 **模组测试** 与 **整机测试** 两种数据源。

### 4.1 访问方式

http://atd.robotera.com:8060/

### 4.2 数据展示

- **表格** ：以表格形式展示测试记录
- **统计** ：总记录数、合格数、不合格数、合格率
- **分页** ：支持分页浏览大量数据

[图片已省略]

### 4.3 筛选

在「筛选条件」区域可按以下条件筛选：

- **SN 编号** ：支持模糊匹配
- **模组型号** （模组测试）/ **测试项名称** （整机测试）：支持筛选
- **测试结果** ：合格 / 不合格
- **日期范围** ：开始日期、结束日期

点击「应用筛选」生效，点击「重置」清除筛选。

### 4.4 切换数据源

- 点击「切换到整机测试」或「切换到模组测试」按钮
- 界面会切换数据源，表格列和筛选条件会相应更新
- 首次打开默认显示模组测试数据

### 4.5 详情、图表、删除（整机测试数据）

[图片已省略]

- **详情** ：点击记录行的「详情」按钮，在弹窗中查看完整信息
- **图表** ：若记录有图表数据（已上传 JSON），会显示「图表」按钮，点击可查看图表（支持缩放）
- **删除** ：点击「删除」按钮，确认后删除该记录，删除后自动刷新

## 5. 常见问题 / 故障排查

### 桌面应用

- **连接失败** ：检查 SSH（默认 192.168.8.100）和 ROS2（ROS_DOMAIN_ID=211）是否可达
- **跨机通讯失败：**表现所有ros控制全部失效，进入nuc的ros2_dev，输入ifconfig出现图所示

### 云端 Web

- **无法加载数据** ：检查 API 服务器是否运行（`curl http://<IP>:8080/api/health`），检查 API 地址是否正确
- **图表无法显示** ：确认记录已上传图表 JSON 文件，格式正确
